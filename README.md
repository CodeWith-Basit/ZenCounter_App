# 🌌 NeonGlow Counter – Premium Flutter UI Showcase

A beautifully crafted, modern take on the classic counter application. Built using the Flutter framework, this project serves as a showcase for high-end UI/UX principles, featuring a stunning dark mode aesthetic, vibrant radial gradient glows, and custom-styled interactive control elements with soft-neon shadow effects.

---

## ✨ Key Features

* **🌌 Premium Dark Aesthetic:** A hand-crafted dark-theme user interface utilizing a rich radial gradient layout that reduces eye strain while delivering a modern, high-tech, and immersive look.
* **🎛️ Advanced Custom Interface:** A perfectly synchronized layout of control actions that bypasses standard platform design constraints for a tailored user experience.
* **💡 Soft Neon Glow Effects:** Implements deep layers of blending and glowing shadows to simulate authentic glowing hardware buttons that react beautifully to the dark backdrop.
* **🎯 Triple-Action State Control:** Integrated states handling precise Increment, Decrement, and instantaneous Value Reset operations smoothly.
* **📱 Custom App Branding:** Fully configured with specialized adaptive app launcher icons for a polished, production-ready feel on mobile devices.

---

## 🛠️ Architecture & UI Concepts

### 1. Ambient Lighting & Depth
The core visual appeal of this application relies on a center-focused lighting effect. Instead of a flat background, the interface simulates depth by emitting a subtle violet glow from the center of the screen, which smoothly bleeds into a deep, rich dark tone toward the corners.

### 2. Ergonomic Controls & Visual Hierarchy
The interactive controls are placed strategically to maximize reachability and aesthetic balance. The primary action button is intentionally enlarged and enhanced with a surrounding neon aura, creating an immediate focal point, while secondary controls support the main layout without causing visual clutter.

## 🚀 Getting Started

### Prerequisites
To run or explore this project locally, you need to have the Flutter SDK installed on your system along with an appropriate development environment (Android Studio, VS Code, or Xcode).

### Running the Project
1. Clone this repository to your local workstation.
2. Open your terminal within the project directory and fetch the required packages using the standard package manager.
3. Launch an emulator or connect a physical testing device.
4. Execute the application using the standard run command to see the fluid transitions and glowing states in action.
